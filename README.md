# IntelSpectra
### AI & ML Enabled Video Analysis and Interpretation
#### Smart India Hackathon (SIH) 2025 Finalist Project

---

## 📌 Overview

IntelSpectra is an AI-powered video intelligence platform developed for the National Security Guard (NSG) under the Ministry of Home Affairs as part of Smart India Hackathon (SIH) 2025.

The system transforms large volumes of surveillance footage into actionable intelligence by automatically detecting suspicious activities, identifying threats, tracking persons and objects, and generating real-time alerts. It significantly reduces manual video analysis time while improving operational efficiency during critical missions.

---

## 🎯 Problem Statement

Security agencies collect enormous amounts of surveillance footage from CCTV cameras, drones, body cameras, and other monitoring systems. Manually reviewing these videos is time-consuming, labor-intensive, and prone to human error.

IntelSpectra addresses this challenge by leveraging Artificial Intelligence and Machine Learning to automate video analysis, enabling security personnel to make faster and more informed decisions.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/IntelSpectra.git
```

Navigate to the project directory

```bash
cd IntelSpectra
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the backend

```bash
python app.py
```

Run the frontend

```bash
npm install
npm start
```
